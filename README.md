# Blinkit-E-Commerce-Project-Using---SQL-Python-Power-Bi-Tableau
📌 Project Overview

This project presents an interactive Power BI dashboard built to analyze Blinkit sales data and uncover meaningful business insights.

The goal of this project is to transform raw data into actionable insights that help understand sales performance, customer preferences, and outlet trends.

📊 Dashboard Features
💰 Total Sales Analysis – Overall revenue performance
📦 Product Insights – Number of items and category distribution
⭐ Customer Ratings – Average rating trends
🏪 Outlet Analysis – Sales by outlet size, type, and location
🥗 Fat Content Analysis – Sales distribution by product type
📈 Trend Analysis – Outlet establishment growth over years

🛠️ Tools & Technologies Used
Power BI – Data visualization and dashboard creation
Data Cleaning – Data preprocessing and transformation
DAX (Data Analysis Expressions) – Calculations and measures

📈 Key Insights
Tier 3 outlets generate the highest sales contribution
Fruits and Snack Foods are top-performing categories
Significant growth spike observed in 2018
Supermarket Type outlets dominate overall sales
Customer ratings remain consistently around 4.0, indicating stable satisfaction
